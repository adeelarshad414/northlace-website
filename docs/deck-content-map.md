# Northlace Modernization Sales Deck Content Map

Source deck: `assets/source/application-modernization-cloud-devops-sales-deck.pdf`

Source inspection notes:

- The committed PDF is 24 slides at 1280 x 720 pt. Its text layer is empty, so source text was extracted from the matching local PPTX file for mapping only.
- The PDF render confirms the original visual theme uses light gray slides, Google-style multi-color accent bars, colored edge stripes, repeated service headers, and cramped process rows. The Northlace output removes those source theme elements.
- Source statistics and proof points remain illustrative until verified. The output keeps explicit TODO markers where client-approved evidence, pricing, ICP, or offer guarantees require human confirmation.

Communication job:
By the end, CTOs, VPs Engineering, platform leaders, and executive sponsors should believe Northlace can turn cloud modernization into a governed 90-day value case because Run, Shield, Ledger, and Shift create one operating standard across delivery, security, cost, and modernization.
